* A dark background with a grid mesh.
* A diffused light effect that follows the cursor.
* Cards with glassmorphism effects, hover shine, and gentle translations.
* Elegant typography using the **Newsreader** font.

Furthermore, the generator supports writing everything in a single Markdown file (like this one) using `## YYYY-MM-DD` headings to separate days, or in separate files inside the `entries/` directory.

Yesterday I was planning the structure of the log. I wanted something extremely simple:

2. Run `npm run build`.
3. Get a single static HTML file ready to view or upload to any server.

No databases, no complex frameworks. Just HTML, quality CSS, and Node.js to compile.
